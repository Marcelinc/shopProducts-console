# Console App reading data from file and displaying in formatted table
Program stworzony został w środowisku node.js w wersji 18.13.0.
Nie są wymagane dodatkowe zależności do instalacji.
Uruchomienie programu polega na wykonaniu w konsoli polecenia node index.js z poziomu utworzonych plików (katalog rozwiazanie1).
